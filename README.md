# Console-Based Instagram Clone in C++

This project is a console-based Instagram clone developed in C++ that simulates core social media functionalities while demonstrating concepts from Programming Fundamentals (PF), Object-Oriented Programming (OOP), and Data Structures (DSA). The project combines real-world social media features with efficient data handling and object-oriented design principles.

## 🚀 Features

### 👤 User Management
- User registration and login system
- Profile creation and management
- Password handling
- User search functionality

### 📸 Posts & Feed
- Create and upload posts
- View user feed
- Like and comment system
- Display post history

### 👥 Social Features
- Follow and unfollow users
- Friend/follower management
- Suggested users system
- User interaction tracking

### 💬 Messaging System
- Direct messaging functionality
- Chat history management
- Notification simulation

### 🔍 Search & Navigation
- Search users and posts
- Menu-driven interface
- Easy console navigation

### 📂 Data Handling
- File handling for persistent storage
- Save and load user data
- Store posts and interactions
